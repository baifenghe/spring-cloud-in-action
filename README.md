### 注册中心地址

~~~
http://localhost:9000/eureka/
~~~




### 模块说明

- 注册中心: registry:9000
- 生产者: provider:9001
- 消费者: consumer:9002
- api网关: api-gateway:9999




- api统一由api-gateway暴露出去
